⚠️ Aviso de Segurança do Windows

O Windows pode mostrar um aviso dizendo que "normalmente não é baixado". Isso é normal porque o PDV Desktop ainda não possui certificado digital (que custa $300-500/ano).

O PDV Desktop é 100% seguro: código aberto, sem conexão com internet, dados salvos localmente, sem telemetria.

Como contornar o aviso:
  1. Clique em "Mais informações" ou "Ver mais"
  2. Clique em "Executar mesmo assim"
  3. Siga o instalador normalmente

Como Instalar

Simples e rápido, em apenas 3 passos

1

Baixe o Instalador

Clique no botão de download acima e salve o arquivo PDV Desktop Setup.exe no seu computador.

2

Execute o Instalador

Dê duplo clique no arquivo baixado. Se aparecer aviso de segurança, clique em "Mais informações" → "Executar mesmo assim".

3

Comece a Usar

Abra o PDV Desktop pelo atalho criado na área de trabalho ou menu iniciar e comece a vender!
